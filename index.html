Travel Partner
By FOT ICT USJ
 
 
 
Travelling by Bus Reinvented!
This is the Travel Partner.
Remember the time you waited for a bus that was already gone? Remember that time when you tried to find the next available bus for your trip? Not anymore.
Introducing the Travel Partner, Where users can easily track their buses location and estimated arrival times.
Features
Travel Partner Mobile Applications are equipped with impressive features to fullfill your requirements while ensuring security
 
Separate Apps
Travel Partner has separate mobile apps for the passengers and buses. Buses can send their location data and the users can track the bus locations of the desired bus.
 
Machine Learning based Arrival Time Prediction
A Machine learning model is used to effectively provide users more accurate estimated arrival time of a bus quickly.
 
Live Location Tracking
You don't need to wait too long for a bus thanks to live location tracking feature in Travel Partner User App.
 
Secure
Travel Partner User App doesn't store user's location data anywhere and ensures data protection.
 
Available for Android and iOS
The Travel Partner Mobile Apps are available for both iOS and Android. So, no need to worry about that.
 
Easy to Pickup Passengers
Bus owners can easily inform their passengers about where they are, and what time the bus is arriving at the passenger's location, which eventually increases passenger count.

Products
We needed to build several applications when developing Travel Partner.
User App
Travel Partner User Mobile Application
Functionality
This App is provided to the Passengers to track buses easily.
Technologies Used
Flutter, Python, Flask, Spring Boot, Google Maps API, Scikit Learn.
Details
Passengers can log into the User App, select the route and destination. After that, selecting the relevant bus will show the current location of the bus with estimate arrival time. No more unwanted waiting at the bus stops!
more
This perfectly fulfills passenger requirements and made aware of the bus locations to increase passenger count for buses. less
See Prototype
Bus App
Travel Partner Bus Mobile Application
Functionality
This App is provided for the bus drivers to share their bus location with passengers.
Technologies Used
Flutter, Python, Flask, Spring Boot, Google Maps API, Scikit Learn.
Details
Bus drivers can easily use The Bus App to select their route, and their destination, so it can be shared with potential passengers.
more
No more slow-tripping across the route and find your passengers even before you see the bus halt! less
See Prototype
The Data Logger App
Data logger App for Data collection
Functionality
The data logger App was used by us to collect bus location data for training ML models.
Technologies Used
Flutter, Python, Flask, Spring Boot, Google Maps API, Scikit Learn.
Description
The data logger App was used by us to collect bus location data that was fed into train the machine learning models. It uses GPS of the mobile phone to get the position and other relevant data.
more
Afte that, the data are sent to the backend and stored as a JSON for further pre-processing and ML model training. less
